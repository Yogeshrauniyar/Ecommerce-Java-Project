# E-Commerce Website

## Project Description

This is a Java-based E-Commerce Website developed to provide an online platform for selling electronic products. The website features user registration, product browsing, and a secure checkout process. Admins can manage products and inventory through a dedicated dashboard.

## Features

* User registration and login functionality
* Browse and search for electronic products
* Add items to the cart and update quantities
* Checkout with a simulated payment process
* Admin panel for adding, updating, and removing products
* Responsive design for seamless use on various devices

## Tech Stack

* Frontend: HTML, CSS, JavaScript, Bootstrap
* Backend: Java (JSP, Servlets), JDBC
* Database: MySQL
* Libraries: Java Mail API for notifications

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/ecommerce-website.git
   ```
2. Import the project into your preferred IDE (e.g., Eclipse).
3. Set up the MySQL database using the provided SQL scripts.
4. Configure database credentials in the project files.
5. Run the project on a local server (e.g., Apache Tomcat).

## Usage

* Access the website at `http://localhost:8080/ecommerce`.
* Register or login as a user.
* Browse products, add items to the cart, and proceed to checkout.
* Admin users can manage products via the admin panel.

## Folder Structure

* `src/main/java`: Contains all Java source files
* `src/main/webapp`: Frontend files (HTML, CSS, JSP)
* `WEB-INF`: Configuration and library files
* `database`: SQL scripts for database setup
